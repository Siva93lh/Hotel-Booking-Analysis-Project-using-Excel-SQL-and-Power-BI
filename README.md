# Hotel-Booking-Analysis-Project-using-Excel-SQL-and-Power-BI
This project analyzes a comprehensive hotel booking dataset to uncover booking trends, guest preferences, cancellation patterns, and factors influencing hotel operations. Using SQL, Excel, and Power BI, the analysis highlights actionable insights that help optimize revenue, improve guest experience, and enhance operational efficiency.
